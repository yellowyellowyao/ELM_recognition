# ELM_recognition
## 背景说明：

ELM（边缘局域模，Edge local mode）是等离子体运行中的一种模式，常伴随H-模（高约束模）一起出现。
ELM出现时，Hα信号幅值整体下降，能量约束水平上升，伴随着大量的准周期性的、快速的尖峰状扰动，即 ELM，如下图所示：
